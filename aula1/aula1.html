1) Os comandos principais:

Navegação:

  • ls (list)
  • cd (change directory)
  • pwd (print work directory)
  • clear

Exemplo de código:

> ls
> cd ~/
> pwd
> ls -l
> cd /home/
> clear

Gerenciando arquivos:

  • mkdir (make directory)
  • touch
  • mv (move)
  • rm (remove)
  • cp (copy)
  • cat (concatenate)
  • head
  • tail
  • comm, cmp (compare)
  • grep

Exemplo de código:

> mkdir teste
> touch teste.txt
> mv teste.txt teste2.txt
> mv ../teste2.txt
> cd ../
> cp teste2.txt teste
> cd teste 
> head teste2.txt
> tail teste2.txt
> touch teste1.txt
> cmp teste1.txt teste2.txt
> grep 'fim' teste2.txt

2) Aplicação:

  • Gerenciando trabalhos (usando while):
#!/bin/bash
if ! [ -e list.txt ]; then
  echo "Could not find a list.txt"
  echo "STOP"
  exit
fi

while read i; do
  if [ -e ../$i/01.out ]
    then
      echo $i
    else
      source job.sh
      sleep 10
      echo $i
  fi
done <list.txt
  • Gerenciando trabalhos (usando for):
#!/bin/bash
n=100
